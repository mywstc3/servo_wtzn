## 2. 板级功能概述

本板是一块 **有刷直流电机 H 桥驱动板**，具备：

- **GD32F130F8P6TR**（TSSOP-20）作为主控
- **AS5600-ASOM** 磁编码器，I2C 读角度
- **EG2104 × 2 + B3202 × 2** 组成 H 桥
- **0.01Ω 采样电阻 + GS8591 运放** 做电流检测
- **ME6119C33** LDO 提供 3.3V 逻辑电

目标应用：位置反馈 + PWM 驱动电机（伺服/调速）。

```text
  P1/P2(VIN) ──► 保护/滤波 ──► VCC ──► LDO ──► 3V3 ──► GD32 + AS5600 + 运放
                    │                              │
                    └──────────────► VC1 ──► EG2104×2 ──► B3202×2 ──► M+/M-
                                                      │
                                                   RS1(0.01Ω) ──► GS8591 ──► MCU ADC
```

## 4. 电源架构

### 4.1 电源网络

| 网络 | 来源 | 用途 |
|------|------|------|
| **VIN** | P1/P2 Pin2 | 电机主电源、板级输入 |
| **VCC** | VIN 经 D4、U4 | LDO 输入、驱动前级 |
| **VC1** | VCC 经 D6 | EG2104 逻辑/自举前级 |
| **3V3** | U5 LDO 输出 | MCU、AS5600 逻辑、运放、I2C 上拉 |
| **GND** | 公共地 | — |

### 4.2 输入路径

```text
P1/P2 Pin2 (VIN)
  ├─ C10 (120nF) + C25 (100µF/6.3V) ── GND
  ├─ D4 (DS14W)
  ├─ U4 (NC 占位/选焊)
  └─ VCC
       ├─ C9 + C11 ── GND
       ├─ U5 ME6119 ──► 3V3
       └─ D6 ──► VC1 ──► U2/U3 VCC
```

### 4.3 ME6119C33M5G 引脚（SOT23-5）

| Pin | 名称 | 连接 |
|-----|------|------|
| 1 | VIN | VCC |
| 2 | VSS | GND |
| 3 | CE | VCC（常使能） |
| 4 | NC | 不连 |
| 5 | VOUT | 3V3 |

### 4.4 AS5600 供电（U8 开关）

- **U8 (2N7002)**：Gate 经 R6(10k) 接 **3V3**，3.3V 正常后导通
- AS5600 **VDD5V (Pin1)** 与 **VDD3V3 (Pin2)** 并联，经 U8 从 VIN 侧供电
- **D5 (1N4148)** 做保护

### 4.5 注意事项

- **C25 额定 6.3V**：VIN 不宜长期超过约 6V（需与实物 BOM 核对）
- **U4 丝印 NC**：可能为不贴或跳线位

## 5. 电机功率级（H 桥）

### 5.1 拓扑

- **U2 + Q1** → 驱动 **M+** 侧半桥
- **U3 + Q2** → 驱动 **M-** 侧半桥
- 两路低端共地回路上串 **RS1 (0.01Ω)**

```text
        VIN
         │
    ┌────┴────┐
   Q1(U2)   Q2(U3)
    │         │
   M+ ── 电机 ── M-
    │         │
    └────┬────┘
         │
       RS1 (0.01Ω)
         │
        GND
```

### 5.2 EG2104 引脚（U2/U3 相同）

| Pin | 名 | 说明 |
|-----|-----|------|
| 1 | VCC | 接 VC1 |
| 2 | IN | PWM 逻辑输入 |
| 3 | #SD | 关断，**低电平关闭驱动** |
| 4 | GND | 地 |
| 5 | LO | 低端 MOS 栅极 |
| 6 | VS | 半桥中点（M+ 或 M-） |
| 7 | HO | 高端 MOS 栅极 |
| 8 | VB | 自举电容 |

- **U2 自举**：D3 + C7 (120nF)
- **U3 自举**：D2 + C8 (120nF)

### 5.3 有刷电机驱动概念

| 动作 | 典型做法 |
|------|----------|
| 正转 | 一侧 IN=PWM（高端斩波），另一侧 IN=0（低端常导通） |
| 反转 | 两侧角色对调 |
| 关断 | #SD 拉低 |
| 使能 | #SD 拉高 |

EG2104 根据 IN 高低在 HO/LO 间互补输出，带死区。

## 7. AS5600 位置传感器

### 7.1 规格书要点

- 12-bit 绝对角度，默认 0~360°
- I2C 7-bit 地址：**0x36**
- I2C 最高 **1 MHz**（Fast-mode Plus）
- 角度寄存器 **ANGLE**：字地址 **0x0E**（高字节），只读，12-bit 有效（0~4095）
- 原始角度 **RAW ANGLE**：字地址 **0x0C**
- 状态寄存器 **STATUS**：字地址 **0x0B**；**MD (bit5)** = 1 表示检测到磁铁

### 7.2 板上连接（硬件固定）

| AS5600 Pin | 名称 | 连接 |
|------------|------|------|
| 1 | VDD5V | 与 Pin2 并联，经 U8 供电 |
| 2 | VDD3V3 | 同上 |
| 3 | OUT | 未使用（走 I2C 即可） |
| 4 | GND | GND |
| 5 | PGO | **3V3**（Programming Option B） |
| 6 | SDA | **PA10**，R15 上拉 3V3 |
| 7 | SCL | **PA9**，R14 上拉 3V3 |
| 8 | DIR | **GND**（顺时针旋转时数值增大） |

### 7.3 使用前提

- 轴上需 **径向充磁磁铁**，与传感器 **同轴**
- 典型气隙约 0.5~3 mm（按 AS5600 规格书）
- 无磁铁时 STATUS.MD = 0

### 7.4 角度换算

```text
角度(度) = raw × 360 / 4096
```

## 11. 信号流（软件开发参考）

```text
         ┌──────── I2C (PA9/PA10) ────────┐
         │                                 │
  VIN ──►│  GD32F130                       ├──► EG2104×2 ──► H桥 ──► M+/M-
         │     ▲                            │
         │     └── ADC: PA3(电流), PA4(VIN)│
         │                                 │
         └──── AS5600 ◄── 磁铁(on-axis)    │
```

推荐控制环：

1. I2C 读 AS5600 ANGLE (0x0E)
2. PID → PWM duty
3. **PA6** 拉高 #SD 使能驱动（低电平关断）
4. 可选：PA3 过流保护，PA4 欠压保护

## 14. 参考链接（规格书内摘要）

### AS5600 I2C

- Slave address: **0x36**
- ANGLE register: **0x0E** (16-bit read, 12-bit valid)
- STATUS register: **0x0B**, MD bit: **0x20**

### EG2104 #SD

- Active **low**；拉高使能，拉低关断所有输出。
