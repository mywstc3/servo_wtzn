## 1. 任务目标

在不依赖闭环控制的前提下，通过占空比/方向直接驱动 H 桥，实现：

- 开环正转 / 反转 / 停止
- 上电默认关断（#SD 拉低）
- PWM 频率约 20 kHz
- 给定固定 duty 时电机稳定转动，无异常发热

**验收标准（来自任务文档）：**

| 检查项 | 预期 |
|--------|------|
| 正转 | PA7 有 PWM，PB1 为低 |
| 反转 | PB1 有 PWM，PA7 为低 |
| 停止 | 两路 PWM compare = 0 |
| #SD 关断 | PA6 拉低后 H 桥无驱动 |
| 频率 | 示波器测 PA7/PB1 约 20 kHz |

## 2. 硬件引脚（原理图 §8.3 为准）

| 信号 | MCU 引脚 | 外设 | 固件配置 |
|------|----------|------|----------|
| U2 IN（M+ 侧 PWM） | **PA7** | U2 EG2104 Pin2 | TIMER2_CH1，AF0 |
| U3 IN（M- 侧 PWM） | **PB1** | U3 EG2104 Pin2 | TIMER2_CH3，AF0 |
| U2/U3 #SD 共用 | **PA6** | U2 Pin3 + U3 Pin3 并联 | GPIO 推挽输出，**低电平关断** |

GPIO / Timer 底层 API 见 [GD32F1x0-外设笔记.md §3、§4](./GD32F1x0-外设笔记.md)。

### 2.1 例程错误对照（必改）

例程 `gpio_output/User/board_config.h` 与原理图不一致：

| 项目 | 例程当前 | 原理图正确 |
|------|----------|------------|
| M+ PWM | PA7 ✓ | PA7 |
| M- PWM | **PA6** ✗ | **PB1** |
| #SD | **PB1** ✗ | **PA6** |

例程把 PA6 当成 TIMER2 第二路 PWM、PB1 当成 #SD，与 EG2104 实际接线完全反了。`motor_set_duty` 等逻辑可参考，**引脚宏和初始化必须重写**。

### 2.2 PB1 定时器复用（M-01-2 结论）

查 GD32F130 数据手册 §2.6.7：

- **PA7 → TIMER2_CH1**，AF0
- **PB1 → TIMER2_CH3**，AF0
- 两路 PWM 共用 **TIMER2**，compare 值可直接对比

## 3. 控制逻辑

### 3.1 EG2104 #SD

- #SD **低电平**：驱动关断，H 桥无输出
- #SD **高电平**：驱动使能，IN 引脚 PWM 才有效
- 上电 / 初始化 / disable 时：PA6 输出低
- enable 时：PA6 输出高

PA6 的 GPIO 写法见 [外设笔记 §3.8](./GD32F1x0-外设笔记.md#38-bsp-示例led-闪烁pf0)（把 LED 换成 #SD 即可）。

### 3.2 正反转策略（互补单路 PWM）

同一时刻只给一侧 IN 加 PWM，另一侧保持低：

```
duty > 0（正转）：PA7 compare = |duty|，PB1 compare = 0
duty < 0（反转）：PB1 compare = |duty|，PA7 compare = 0
duty = 0（停止）：PA7 compare = 0，PB1 compare = 0
```

`motor_disable()` 应先 `motor_set_duty(0)` 再拉低 #SD，避免关断瞬间仍有 PWM 残留。

### 3.3 duty 取值范围

- 类型：`int16_t`，正负表示方向
- 幅度上限：`±MOTOR_PWM_PERIOD`
- 默认 `MOTOR_PWM_PERIOD = 3599`（与 20 kHz 对应，见 §4）

---

## 4. PWM 频率（本板参数）

目标 20 kHz，TIMER2 挂在 APB1：

```
f_pwm = f_timer / ((PSC + 1) × (ARR + 1))
```

本板取 `PSC = 0`，`ARR = 3599` → 分频系数 3600。  
若 TIMER2 时钟为 72 MHz：`72000000 / 3600 = 20000 Hz`。

以 `system_gd32f1x0.c` 实际时钟为准，频率不对时用示波器微调 ARR/PSC。  
Timer 通用配置方法见 [外设笔记 §4](./GD32F1x0-外设笔记.md#4-timerpwm)。

## 5. 实现步骤（按子任务顺序）

### 5.1 M-01-1：新建 `board_config.h`

路径：`servo_project/User/board_config.h`

```c
#ifndef BOARD_CONFIG_H
#define BOARD_CONFIG_H

#include "gd32f1x0.h"

/* M+ 侧 PWM：PA7 = TIMER2_CH1 */
#define MOTOR_PWM_FWD_PORT        GPIOA
#define MOTOR_PWM_FWD_PIN         GPIO_PIN_7
#define MOTOR_PWM_FWD_CH          TIMER_CH_1

/* M- 侧 PWM：PB1 = TIMER2_CH3 */
#define MOTOR_PWM_REV_PORT        GPIOB
#define MOTOR_PWM_REV_PIN         GPIO_PIN_1
#define MOTOR_PWM_REV_CH          TIMER_CH_3

#define MOTOR_PWM_TIMER           TIMER2
#define MOTOR_PWM_TIMER_RCU       RCU_TIMER2
#define MOTOR_PWM_TIMER_AF        GPIO_AF_0

/* #SD：PA6，低电平关断 */
#define MOTOR_SD_PORT             GPIOA
#define MOTOR_SD_PIN              GPIO_PIN_6

#define MOTOR_PWM_FREQ_HZ         20000U
#define MOTOR_PWM_PERIOD          3599U

#endif
```

### 5.2 M-01-2 ~ M-01-5：新建 `motor.c` / `motor.h`

路径：`servo_project/User/motor.c`、`motor.h`（或 `servo/motor.c`）

**接口：**

```c
void motor_init(void);
void motor_enable(void);
void motor_disable(void);
void motor_set_duty(int16_t duty);
```

**`motor_init()` 流程：**

1. 使能时钟：`RCU_GPIOA`、`RCU_GPIOB`、`RCU_TIMER2`
2. PA7、PB1：AF0 + 复用推挽 + 50 MHz（见 [外设笔记 §3.6](./GD32F1x0-外设笔记.md#36-复用输出af如-pwm--i2c)）
3. PA6：普通 GPIO 推挽，初始低电平
4. 配置 TIMER2：period = MOTOR_PWM_PERIOD，prescaler = 0
5. 配置 CH1（PA7）和 CH3（PB1）为 PWM0，compare = 0，启动定时器

**相对例程的关键改动：**

| 例程写法 | 应改为 |
|----------|--------|
| `MOTOR_PWM_B` = PA6, `TIMER_CH_0` | PB1, `TIMER_CH_3` |
| `MOTOR_SD` = PB1 | PA6 |
| `gpio_af_set` 只配 PA7/PA6 | 配 PA7 + PB1 |
| `timer_channel_output_config` CH0+CH1 | CH1 + CH3 |

**`motor_set_duty()`：** 限幅 ±PERIOD；duty≥0 时 CH1=duty、CH3=0；duty<0 时相反。  
例程 `motor.c` 第 72–91 行逻辑可照搬，仅替换通道宏。

### 5.3 M-01-6：Keil 集成 + 开环测试

1. Keil User 组添加 `board_config.h`、`motor.c`、`motor.h`
2. `main.c` 最小测试：

```c
#include "gd32f1x0.h"
#include "systick.h"
#include "motor.h"

#define TEST_DUTY    1200
#define RUN_MS       2000U

int main(void)
{
    systick_config();
    motor_init();
    delay_1ms(100U);
    motor_enable();

    while (1) {
        motor_set_duty(TEST_DUTY);
        delay_1ms(RUN_MS);
        motor_set_duty(0);
        delay_1ms(500U);
        motor_set_duty(-TEST_DUTY);
        delay_1ms(RUN_MS);
        motor_set_duty(0);
        delay_1ms(500U);
    }
}
```

**安全：** 首次 duty 从小值（如 500）开始；确认 `#SD` disable 后立刻停转。

## 6. 调试与排错

| 现象 | 排查 |
|------|------|
| 无波形 | 是否 `motor_enable()`；PA7/PB1 AF0；TIMER2 是否启动；通道是否 CH1/CH3 |
| 只一侧转 | 检查 PB1 波形；duty 正负是否反 |
| 频率不对 | 查 `SystemCoreClock`，重算 ARR/PSC |
| 电机发热 | 降低 duty；检查两路 PWM 互斥；PA6 默认是否低 |
| 例程能转但正式工程不行 | 例程引脚错误，必须以 §8.3 为准 |
