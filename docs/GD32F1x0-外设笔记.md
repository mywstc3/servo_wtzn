## 1. 库文件与头文件入口

### 1.1 文件位置

| 层级 | 路径 | 作用 |
|------|------|------|
| 统一入口 | `User/gd32f1x0.h`（通过 Keil Pack） | 芯片寄存器定义 |
| 可选包含 | `User/gd32f1x0_libopt.h` | 按需 `#include` 各外设头文件 |
| 外设声明 | `Library/Include/gd32f1x0_*.h` | API 原型、宏、枚举 |
| 外设实现 | `Library/Source/gd32f1x0_*.c` | 库函数实现（一般不改） |

`gd32f1x0_libopt.h` 已包含 GPIO、RCU、Timer、I2C、USART、ADC 等头文件。  
用户代码 **只 `#include "gd32f1x0.h"`** 即可，不要单独 `#include "gd32f1x0_gpio.h"`。

> **clangd 提示：** `gd32f1x0_gpio.h` 内部会再包含 `gd32f1x0.h`，而 `gd32f1x0.h` 又通过 `gd32f1x0_libopt.h` 回头包含 `gd32f1x0_gpio.h`。在用户头文件里直接 include `gd32f1x0_gpio.h` 会触发 clangd 报 `main file cannot be included recursively`。Keil 编译通常仍能通过，但 IDE 会误报。

### 1.2 代码分层建议

```text
servo_project/User/
├── board_config.h          # 引脚宏（只定义，不写逻辑）
├── bsp/
│   ├── bsp_gpio.c/h        # GPIO 封装
│   ├── bsp_pwm.c/h         # Timer PWM 封装
│   ├── bsp_i2c.c/h         # I2C 封装（待建）
│   └── ...
├── servo/                  # 应用层（motor、pid 等）
└── main.c                  # 只调 bsp / 应用接口
```

**原则：** `main.c` 不直接操作寄存器；底层调用 `Library` 里的标准库函数；引脚定义集中在 `board_config.h`。

## 3. GPIO

### 3.1 相关文件

| 文件 | 路径 |
|------|------|
| 头文件（查 API） | [`Library/Include/gd32f1x0_gpio.h`](../servo_project/Library/Include/gd32f1x0_gpio.h) |

### 3.2 常用 API（`gd32f1x0_gpio.h` 约 365–390 行）

| 函数 | 作用 |
|------|------|
| `gpio_mode_set(port, mode, pull, pin)` | 设输入 / 输出 / 复用 / 模拟 |
| `gpio_output_options_set(port, otype, speed, pin)` | 推挽/开漏、输出速度 |
| `gpio_af_set(port, af, pin)` | 复用功能编号（PWM、I2C 等） |
| `gpio_bit_set(port, pin)` | 输出高 |
| `gpio_bit_reset(port, pin)` | 输出低 |
| `gpio_bit_write(port, pin, value)` | 写指定位 |
| `gpio_input_bit_get(port, pin)` | 读输入电平 |

### 3.3 常用宏

**模式（`gpio_mode_set` 第 2 参数）：**

| 宏 | 含义 |
|----|------|
| `GPIO_MODE_OUTPUT` | 普通输出 |
| `GPIO_MODE_INPUT` | 输入 |
| `GPIO_MODE_AF` | 复用（Timer、I2C、USART） |
| `GPIO_MODE_ANALOG` | 模拟（ADC） |

**上下拉（第 3 参数）：**

| 宏 | 含义 |
|----|------|
| `GPIO_PUPD_NONE` | 浮空 |
| `GPIO_PUPD_PULLUP` | 上拉 |
| `GPIO_PUPD_PULLDOWN` | 下拉 |

**输出类型与速度（`gpio_output_options_set`）：**

| 宏 | 含义 |
|----|------|
| `GPIO_OTYPE_PP` | 推挽 |
| `GPIO_OTYPE_OD` | 开漏 |
| `GPIO_OSPEED_2MHZ` | 2 MHz（#SD、LED 等） |
| `GPIO_OSPEED_50MHZ` | 50 MHz（PWM 等） |

**端口与引脚：**

```c
GPIOA, GPIOB, GPIOF          /* 端口 */
GPIO_PIN_0 ~ GPIO_PIN_15      /* 位掩码，可 OR 组合 */
GPIO_AF_0 ~ GPIO_AF_11       /* 复用编号，查数据手册 AF 表 */
```

### 3.4 普通 GPIO 输出 — 标准四步

```c
rcu_periph_clock_enable(RCU_GPIOx);                                    /* ① 开时钟 */
gpio_mode_set(GPIOx, GPIO_MODE_OUTPUT, GPIO_PUPD_NONE, GPIO_PIN_n);   /* ② 模式 */
gpio_output_options_set(GPIOx, GPIO_OTYPE_PP, GPIO_OSPEED_2MHZ, GPIO_PIN_n); /* ③ 输出属性 */
gpio_bit_set(GPIOx, GPIO_PIN_n);   /* 或 gpio_bit_reset — ④ 电平 */
```

### 3.5 普通 GPIO 输入

```c
rcu_periph_clock_enable(RCU_GPIOx);
gpio_mode_set(GPIOx, GPIO_MODE_INPUT, GPIO_PUPD_PULLUP, GPIO_PIN_n);

if (gpio_input_bit_get(GPIOx, GPIO_PIN_n) == SET) {
    /* 高电平 */
}
```

### 3.6 复用输出（AF，如 PWM / I2C）

与普通 GPIO 不同，需要额外一步：

```c
rcu_periph_clock_enable(RCU_GPIOA);
gpio_af_set(GPIOA, GPIO_AF_0, GPIO_PIN_7);                           /* ① AF 编号 */
gpio_mode_set(GPIOA, GPIO_MODE_AF, GPIO_PUPD_NONE, GPIO_PIN_7);      /* ② 复用模式 */
gpio_output_options_set(GPIOA, GPIO_OTYPE_PP, GPIO_OSPEED_50MHZ, GPIO_PIN_7);
/* ③ 还需配置对应 Timer / I2C / USART 外设 */
```

### 3.7 本板 GPIO 引脚速查

| 引脚 | 功能 | GPIO 模式 |
|------|------|-----------|
| PF0 | LED1 | `GPIO_MODE_OUTPUT` 推挽 |
| PA6 | EG2104 #SD | `GPIO_MODE_OUTPUT` 推挽，低关断 |
| PA7 | M+ PWM | `GPIO_MODE_AF` + TIMER2_CH1 |
| PB1 | M- PWM | `GPIO_MODE_AF` + TIMER2_CH3 |
| PA9/PA10 | AS5600 I2C | `GPIO_MODE_AF` + AF4 |
| PA3/PA4 | ADC 采样 | `GPIO_MODE_ANALOG` |

引脚详情见 [硬件说明 §8](./硬件说明-16999-PS26040802.md)。

### 3.8 BSP 示例：LED 闪烁（PF0）

**`board_config.h`：**

```c
#define LED_PORT    GPIOF
#define LED_PIN     GPIO_PIN_0
```

**`bsp_gpio.c`：**

```c
#include "bsp_gpio.h"
#include "gd32f1x0_rcu.h"

void bsp_gpio_init(void)
{
    rcu_periph_clock_enable(RCU_GPIOF);
    gpio_mode_set(LED_PORT, GPIO_MODE_OUTPUT, GPIO_PUPD_NONE, LED_PIN);
    gpio_output_options_set(LED_PORT, GPIO_OTYPE_PP, GPIO_OSPEED_2MHZ, LED_PIN);

```

**`main.c`：**

```c
bsp_gpio_init();
while (1) {
    bsp_led_on();  delay_1ms(500);
    bsp_led_off(); delay_1ms(500);
}
```

## 4. Timer（PWM）

PWM 由 **定时器时基**（PSC、ARR）+ **通道输出比较**（compare）共同决定。  
GPIO 复用部分见 [§3.6](#36-复用输出af如-pwm--i2c)；电机正反转、duty 范围、#SD 使能见 [M-01-电机驱动实现笔记.md §3–§5](./M-01-电机驱动实现笔记.md)。  
电流环同步采样（中心对齐 + TRGO 触发 ADC）见 [§4.10](#410-中心对齐-pwm--trgo-触发-adc电流环同步)。

### 4.1 相关文件

| 文件 | 路径 |
|------|------|
| 头文件（查 API） | [`Library/Include/gd32f1x0_timer.h`](../servo_project/Library/Include/gd32f1x0_timer.h) |
| 源文件（实现） | [`Library/Source/gd32f1x0_timer.c`](../servo_project/Library/Source/gd32f1x0_timer.c) |

与 GPIO 相同，用户代码 **只 `#include "gd32f1x0.h"`**，不要单独 include `gd32f1x0_timer.h`。

### 4.2 频率与占空比

TIMER2 挂在 **APB1**。GD32 在 APB 预分频 ≠ 1 时，定时器时钟通常 **×2**；本工程 `SystemCoreClock = 72 MHz` 且 APB1 常为 ÷2 时，TIMER2 输入时钟一般为 **72 MHz**（以 `system_gd32f1x0.c` 实际配置为准）。

```
f_pwm = f_timer / ((PSC + 1) × (ARR + 1))
占空比 = compare / (ARR + 1)
```

本板目标 **20 kHz**，取 `PSC = 0`、`ARR = 3599`（分频系数 3600）：

```
72000000 / 3600 = 20000 Hz
```

compare 取值范围 `0 ~ ARR`（即 `0 ~ 3599`）。频率不对时用示波器微调 ARR/PSC。

### 4.3 常用 API

**时基：**

| 函数 | 作用 |
|------|------|
| `timer_deinit(timer)` | 复位定时器（初始化前可选调用） |
| `timer_struct_para_init(&para)` | 填充时基结构体默认值 |
| `timer_init(timer, &para)` | 配置 PSC、ARR、计数方向 |
| `timer_enable(timer)` | 启动计数 |
| `timer_disable(timer)` | 停止计数 |

**通道输出（PWM）：**

| 函数 | 作用 |
|------|------|
| `timer_channel_output_struct_para_init(&para)` | 填充通道结构体默认值 |
| `timer_channel_output_config(timer, ch, &para)` | 配置通道输出属性 |
| `timer_channel_output_mode_config(timer, ch, mode)` | 设为 PWM 模式（如 `TIMER_OC_MODE_PWM0`） |
| `timer_channel_output_pulse_value_config(timer, ch, pulse)` | 写 compare（占空比） |
| `timer_channel_output_shadow_config(timer, ch, shadow)` | 是否启用 compare 影子寄存器 |

### 4.4 常用宏

**定时器与通道：**

```c
TIMER0, TIMER1, TIMER2, TIMER13, ...   /* 定时器实例 */
TIMER_CH_0 ~ TIMER_CH_3               /* 通道编号 */
RCU_TIMER2                            /* TIMER2 时钟（见 §2.3） */
```

**时基（`timer_init` 的 `timer_parameter_struct`）：**

| 宏 | 含义 |
|----|------|
| `TIMER_COUNTER_EDGE` | 边沿对齐（普通 PWM 用这个） |
| `TIMER_COUNTER_UP` | 向上计数 |
| `TIMER_CKDIV_DIV1` | 时钟不分频 |

**通道输出（`timer_oc_parameter_struct` + 模式）：**

| 宏 | 含义 |
|----|------|
| `TIMER_CCX_ENABLE` | 使能通道输出 |
| `TIMER_CCXN_DISABLE` | 禁用互补输出 |
| `TIMER_OC_POLARITY_HIGH` | 有效电平为高 |
| `TIMER_OC_MODE_PWM0` | PWM 模式 0（计数 < compare 为高） |
| `TIMER_OC_SHADOW_DISABLE` | compare 立即生效 |

> **说明：** `TIMER_OC_MODE_PWM0` 下，compare = 0 时输出全程低；compare = ARR 时接近 100% 高。  
> TIMER2 为通用定时器，**不需要** `timer_primary_output_config()` 等高级定时器刹车/互补配置。

### 4.5 本板 Timer 复用速查

查 GD32F130 数据手册 §2.6.7：

| 引脚 | 功能 | Timer 通道 | AF |
|------|------|------------|-----|
| PA7 | M+ PWM | TIMER2 **CH1** | AF0 |
| PB1 | M- PWM | TIMER2 **CH3** | AF0 |

两路 PWM **共用 TIMER2**，compare 值可直接对比。

> **例程陷阱：** `gpio_output` 例程把 PA6 当第二路 PWM、PB1 当 #SD，与原理图相反。必须以 [硬件说明 §8.3](./硬件说明-16999-PS26040802.md) 为准：**PA7=CH1，PB1=CH3，PA6=#SD**。

### 4.6 PWM 输出 — 标准六步

与 GPIO「开时钟 → 设模式 → 写电平」对应：

```text
① 开时钟（GPIO + TIMER）
② GPIO 复用（AF → MODE_AF → 50 MHz 推挽，见 §3.6）
③ 定时器时基（PSC、ARR、向上计数）
④ 通道输出（PWM0 模式、极性、使能）
⑤ compare = 0（上电无输出）
⑥ timer_enable() 启动
```

```c
rcu_periph_clock_enable(RCU_GPIOA);
rcu_periph_clock_enable(RCU_GPIOB);
rcu_periph_clock_enable(RCU_TIMER2);                              /* ① */

gpio_af_set(GPIOA, GPIO_AF_0, GPIO_PIN_7);                        /* ② */
gpio_af_set(GPIOB, GPIO_AF_0, GPIO_PIN_1);
gpio_mode_set(GPIOA, GPIO_MODE_AF, GPIO_PUPD_NONE, GPIO_PIN_7);
gpio_mode_set(GPIOB, GPIO_MODE_AF, GPIO_PUPD_NONE, GPIO_PIN_1);
gpio_output_options_set(GPIOA, GPIO_OTYPE_PP, GPIO_OSPEED_50MHZ, GPIO_PIN_7);
gpio_output_options_set(GPIOB, GPIO_OTYPE_PP, GPIO_OSPEED_50MHZ, GPIO_PIN_1);

timer_parameter_struct timer_initpara;
timer_oc_parameter_struct timer_ocinitpara;

timer_deinit(TIMER2);                                             /* ③ */
timer_struct_para_init(&timer_initpara);
timer_initpara.prescaler         = 0U;
timer_initpara.alignedmode       = TIMER_COUNTER_EDGE;
timer_initpara.counterdirection  = TIMER_COUNTER_UP;
timer_initpara.period            = 3599U;   /* ARR */
timer_initpara.clockdivision     = TIMER_CKDIV_DIV1;
timer_init(TIMER2, &timer_initpara);

timer_channel_output_struct_para_init(&timer_ocinitpara);        /* ④ */
timer_ocinitpara.outputstate  = TIMER_CCX_ENABLE;
timer_ocinitpara.outputnstate = TIMER_CCXN_DISABLE;
timer_ocinitpara.ocpolarity   = TIMER_OC_POLARITY_HIGH;
timer_ocinitpara.ocnpolarity  = TIMER_OCN_POLARITY_HIGH;
timer_ocinitpara.ocidlestate  = TIMER_OC_IDLE_STATE_LOW;
timer_ocinitpara.ocnidlestate = TIMER_OCN_IDLE_STATE_LOW;

timer_channel_output_config(TIMER2, TIMER_CH_1, &timer_ocinitpara);
timer_channel_output_config(TIMER2, TIMER_CH_3, &timer_ocinitpara);
timer_channel_output_mode_config(TIMER2, TIMER_CH_1, TIMER_OC_MODE_PWM0);
timer_channel_output_mode_config(TIMER2, TIMER_CH_3, TIMER_OC_MODE_PWM0);
timer_channel_output_shadow_config(TIMER2, TIMER_CH_1, TIMER_OC_SHADOW_DISABLE);
timer_channel_output_shadow_config(TIMER2, TIMER_CH_3, TIMER_OC_SHADOW_DISABLE);

timer_channel_output_pulse_value_config(TIMER2, TIMER_CH_1, 0U);  /* ⑤ */
timer_channel_output_pulse_value_config(TIMER2, TIMER_CH_3, 0U);

timer_enable(TIMER2);                                             /* ⑥ */
```

运行时改占空比只需调用 `timer_channel_output_pulse_value_config()`，不必重启定时器。

### 4.7 BSP 示例：`bsp_pwm.c/h`

**`board_config.h`：**

```c
#define MOTOR_PWM_FWD_PORT        GPIOA
#define MOTOR_PWM_FWD_PIN         GPIO_PIN_7
#define MOTOR_PWM_FWD_CH          TIMER_CH_1

#define MOTOR_PWM_REV_PORT        GPIOB
#define MOTOR_PWM_REV_PIN         GPIO_PIN_1
#define MOTOR_PWM_REV_CH          TIMER_CH_3

#define MOTOR_PWM_TIMER           TIMER2
#define MOTOR_PWM_TIMER_RCU       RCU_TIMER2
#define MOTOR_PWM_TIMER_AF        GPIO_AF_0

#define MOTOR_PWM_FREQ_HZ         20000U
#define MOTOR_PWM_PERIOD          3599U   /* ARR，20 kHz @ 72 MHz TIMER 时钟 */
```

**`bsp_pwm.h`：**

```c
#ifndef BSP_PWM_H
#define BSP_PWM_H
#include "gd32f1x0.h"

void bsp_pwm_init(void);
void bsp_pwm_set_duty(int16_t duty);   /* ±MOTOR_PWM_PERIOD，正负表方向 */

#endif
```

**`bsp_pwm.c`（核心逻辑）：**

```c
#include "bsp_pwm.h"
#include "board_config.h"

static void pwm_set_compare(uint16_t channel, uint16_t pulse)
{
    if (pulse > MOTOR_PWM_PERIOD) {
        pulse = MOTOR_PWM_PERIOD;
    }
    timer_channel_output_pulse_value_config(MOTOR_PWM_TIMER, channel, pulse);
}

void bsp_pwm_init(void)
{
    timer_parameter_struct timer_initpara;
    timer_oc_parameter_struct timer_ocinitpara;

    rcu_periph_clock_enable(RCU_GPIOA);
    rcu_periph_clock_enable(RCU_GPIOB);
    rcu_periph_clock_enable(MOTOR_PWM_TIMER_RCU);

    gpio_af_set(MOTOR_PWM_FWD_PORT, MOTOR_PWM_TIMER_AF, MOTOR_PWM_FWD_PIN);
    gpio_af_set(MOTOR_PWM_REV_PORT, MOTOR_PWM_TIMER_AF, MOTOR_PWM_REV_PIN);
    gpio_mode_set(MOTOR_PWM_FWD_PORT, GPIO_MODE_AF, GPIO_PUPD_NONE, MOTOR_PWM_FWD_PIN);
    gpio_mode_set(MOTOR_PWM_REV_PORT, GPIO_MODE_AF, GPIO_PUPD_NONE, MOTOR_PWM_REV_PIN);
    gpio_output_options_set(MOTOR_PWM_FWD_PORT, GPIO_OTYPE_PP, GPIO_OSPEED_50MHZ, MOTOR_PWM_FWD_PIN);
    gpio_output_options_set(MOTOR_PWM_REV_PORT, GPIO_OTYPE_PP, GPIO_OSPEED_50MHZ, MOTOR_PWM_REV_PIN);

    timer_deinit(MOTOR_PWM_TIMER);
    timer_struct_para_init(&timer_initpara);
    timer_initpara.prescaler         = 0U;
    timer_initpara.alignedmode       = TIMER_COUNTER_EDGE;
    timer_initpara.counterdirection  = TIMER_COUNTER_UP;
    timer_initpara.period            = MOTOR_PWM_PERIOD;
    timer_initpara.clockdivision     = TIMER_CKDIV_DIV1;
    timer_init(MOTOR_PWM_TIMER, &timer_initpara);

    timer_channel_output_struct_para_init(&timer_ocinitpara);
    timer_ocinitpara.outputstate  = TIMER_CCX_ENABLE;
    timer_ocinitpara.outputnstate = TIMER_CCXN_DISABLE;
    timer_ocinitpara.ocpolarity   = TIMER_OC_POLARITY_HIGH;
    timer_ocinitpara.ocnpolarity  = TIMER_OCN_POLARITY_HIGH;
    timer_ocinitpara.ocidlestate  = TIMER_OC_IDLE_STATE_LOW;
    timer_ocinitpara.ocnidlestate = TIMER_OCN_IDLE_STATE_LOW;

    timer_channel_output_config(MOTOR_PWM_TIMER, MOTOR_PWM_FWD_CH, &timer_ocinitpara);
    timer_channel_output_config(MOTOR_PWM_TIMER, MOTOR_PWM_REV_CH, &timer_ocinitpara);
    timer_channel_output_mode_config(MOTOR_PWM_TIMER, MOTOR_PWM_FWD_CH, TIMER_OC_MODE_PWM0);
    timer_channel_output_mode_config(MOTOR_PWM_TIMER, MOTOR_PWM_REV_CH, TIMER_OC_MODE_PWM0);
    timer_channel_output_shadow_config(MOTOR_PWM_TIMER, MOTOR_PWM_FWD_CH, TIMER_OC_SHADOW_DISABLE);
    timer_channel_output_shadow_config(MOTOR_PWM_TIMER, MOTOR_PWM_REV_CH, TIMER_OC_SHADOW_DISABLE);

    pwm_set_compare(MOTOR_PWM_FWD_CH, 0U);
    pwm_set_compare(MOTOR_PWM_REV_CH, 0U);
    timer_enable(MOTOR_PWM_TIMER);
}

void bsp_pwm_set_duty(int16_t duty)
{
    uint16_t mag;

    if (duty > (int16_t)MOTOR_PWM_PERIOD) {
        duty = (int16_t)MOTOR_PWM_PERIOD;
    } else if (duty < -(int16_t)MOTOR_PWM_PERIOD) {
        duty = -(int16_t)MOTOR_PWM_PERIOD;
    }

    if (duty >= 0) {
        mag = (uint16_t)duty;
        pwm_set_compare(MOTOR_PWM_FWD_CH, mag);
        pwm_set_compare(MOTOR_PWM_REV_CH, 0U);
    } else {
        mag = (uint16_t)(-duty);
        pwm_set_compare(MOTOR_PWM_FWD_CH, 0U);
        pwm_set_compare(MOTOR_PWM_REV_CH, mag);
    }
}
```

`bsp_pwm_set_duty()` 的互斥逻辑（同一时刻只给一侧 IN 加 PWM）属于电机应用策略，详见 [M-01 §3.2](./M-01-电机驱动实现笔记.md#32-正反转策略互补单路-pwm)。  
#SD（PA6）为普通 GPIO，不在 `bsp_pwm` 里配置，由 `motor.c` 或单独 BSP 管理。

**`main.c` 最小调用：**

```c
#include "bsp_pwm.h"
#include "systick.h"

int main(void)
{
    systick_config();
    bsp_pwm_init();

    while (1) {
        bsp_pwm_set_duty(1200);   /* 正转 */
        delay_1ms(2000);
        bsp_pwm_set_duty(0);      /* 停止 */
        delay_1ms(500);
        bsp_pwm_set_duty(-1200);  /* 反转 */
        delay_1ms(2000);
        bsp_pwm_set_duty(0);
        delay_1ms(500);
    }
}
```

| 现象 | 排查 |
|------|------|
| 无波形 | 是否 `timer_enable()`；PA7/PB1 是否 AF0；通道是否为 **CH1/CH3**（不是例程的 CH0/CH1） |

### 4.10 中心对齐 PWM + TRGO 触发 ADC（电流环同步）

开环阶段 §4.6 的 **边沿对齐** 足够。做 **电流闭环** 时，建议改为 **中心对齐 PWM**，并由 **TIMER2 TRGO** 触发 ADC 采样（ADC 侧配置见 [§7.12](#712-timer-触发--dma--中断完整链路)）。

#### 4.10.1 信号链路

```text
TIMER2（中心对齐 PWM，持续运行）
    │ TRGO（Update 事件）
    ▼
ADC 规则组扫描（CH3 电流 + CH4 电压）
    │ 转换完成
    ├── DMA ──► 内存缓冲 [I_raw, V_raw, ...]
    └── ADC EOC 中断 ──► 读缓冲 / 电流环 ──► 写 TIMER2 CCR（下一周期生效）
```

**要点：**

- **PWM 触发 ADC**，不是 ADC 触发 PWM 时基；TIMER 自由运行，ISR 只更新 **CCR**。
- 采样时刻应落在 PWM **谷底/顶点**（电流较稳），中心对齐比边沿对齐更合适。
- CCR 建议开 **影子寄存器**，在 UPDATE 事件加载，避免 duty 毛刺。

#### 4.10.2 中心对齐频率公式

```
f_pwm = f_timer / (2 × (PSC + 1) × (ARR + 1))    /* 中心对齐 */
占空比 = compare / ARR                            /* 中心对齐时 compare 相对 ARR */
```

目标 **20 kHz**、TIMER 时钟 **72 MHz**、PSC = 0：

```
ARR = 72_000_000 / (2 × 20_000) - 1 = 1799
```

compare 范围 **0 ~ ARR**（0 ~ 1799）。与边沿对齐 §4.2 的 ARR = 3599 **不可混用**。

#### 4.10.3 额外 API（主模式 / 影子）

| 函数 | 作用 |
|------|------|
| `timer_master_output_trigger_source_select(timer, src)` | 选择 **TRGO** 输出源 |
| `timer_auto_reload_shadow_enable(timer)` | ARR 影子寄存器 |
| `timer_channel_output_shadow_config(timer, ch, shadow)` | CCR 影子寄存器 |

#### 4.10.4 额外宏

**中心对齐（`timer_init` → `alignedmode`）：**

| 宏 | 含义 |
|----|------|
| `TIMER_COUNTER_EDGE` | 边沿对齐（§4.6 开环用这个） |
| `TIMER_COUNTER_CENTER_DOWN` | 中心对齐，向下计数比较有效（**常用**） |
| `TIMER_COUNTER_CENTER_UP` | 中心对齐，向上计数比较有效 |
| `TIMER_COUNTER_CENTER_BOTH` | 中心对齐，上下均有效 |

**TRGO 源（`timer_master_output_trigger_source_select` 第 2 参数）：**

| 宏 | 含义 |
|----|------|
| `TIMER_TRI_OUT_SRC_UPDATE` | **更新事件**作 TRGO（与 ADC 同步最常用） |
| `TIMER_TRI_OUT_SRC_ENABLE` | CEN 使能信号 |
| `TIMER_TRI_OUT_SRC_CH0` ~ `O3CPRE` | 通道比较事件 |

本板选 **`TIMER_TRI_OUT_SRC_UPDATE`**：计数器回零/换向时触发 ADC，对应 PWM 周期锚点。

#### 4.10.5 TIMER2 侧 — 配置步骤

在 §4.6 六步基础上 **改动以下项**：

```text
① 时基改为 center-aligned，ARR 按 §4.10.2 重算（1799 @ 20 kHz）
② timer_master_output_trigger_source_select(TIMER2, TIMER_TRI_OUT_SRC_UPDATE)
③ timer_auto_reload_shadow_enable(TIMER2)
④ 两路 CCR 开影子：timer_channel_output_shadow_config(..., TIMER_OC_SHADOW_ENABLE)
⑤ PWM 通道配置不变（CH1/CH3、PWM0）
⑥ timer_enable() 后 TIMER2 即按 TRGO 周期触发 ADC（ADC 侧须已配外部触发）
```

```c
timer_parameter_struct timer_initpara;
timer_oc_parameter_struct timer_ocinitpara;

timer_deinit(TIMER2);
timer_struct_para_init(&timer_initpara);
timer_initpara.prescaler         = 0U;
timer_initpara.alignedmode       = TIMER_COUNTER_CENTER_DOWN;  /* 中心对齐 */
timer_initpara.counterdirection  = TIMER_COUNTER_UP;
timer_initpara.period            = 1799U;                      /* 20 kHz @ 72 MHz */
timer_initpara.clockdivision     = TIMER_CKDIV_DIV1;
timer_init(TIMER2, &timer_initpara);

/* ... CH1/CH3 PWM 配置同 §4.6 ... */

timer_master_output_trigger_source_select(TIMER2, TIMER_TRI_OUT_SRC_UPDATE);
timer_auto_reload_shadow_enable(TIMER2);
timer_channel_output_shadow_config(TIMER2, TIMER_CH_1, TIMER_OC_SHADOW_ENABLE);
timer_channel_output_shadow_config(TIMER2, TIMER_CH_3, TIMER_OC_SHADOW_ENABLE);

timer_enable(TIMER2);
```

**ISR 中更新 duty：** 只调 `timer_channel_output_pulse_value_config()` 写 CCR；影子寄存器会在下一次 UPDATE 生效。

#### 4.10.6 调试速查（PWM + ADC 同步）

| 现象 | 排查 |
|------|------|
| ADC 不采样 | ADC 是否 `adc_external_trigger_config(ENABLE)` 且源为 `T2_TRGO`；TIMER2 是否已 enable |
| 采样频率不对 | 中心对齐 ARR 是否按 §4.10.2 重算；TRGO 是否为 UPDATE |
| duty 更新有毛刺 | CCR 是否开影子；ISR 中勿 `timer_enable/disable` |
| 电流采样噪声大 | TRGO 是否在开关边沿；可试 `CENTER_DOWN` 或示波器对齐 TRGO 与 PWM |

### 7.2 基本参数（GD32F130）

| 项目 | 说明 |
|------|------|
| 分辨率 | **12-bit**，右对齐时结果 **0 ~ 4095** |

### 7.6 引脚与通道对应（GD32F130）

PA 口与 ADC 通道一一对应：**PAx → ADC_INx → ADC_CHANNEL_x**。  
配置 GPIO 时用 `GPIO_MODE_ANALOG`，**不要**设为 AF 或数字输入。

本板（16999-PS26040802）：

| 引脚 | 信号 | 硬件来源 |
|------|------|----------|
| PA3 | 母线电流采样 | GS8591 运放输出 → R11/C15 滤波 |
| PA4 | 母线电压采样 | VIN 分压 + 钳位 |

> **注意：** PA4 在本板是 **VIN ADC**，不是 AS5600 DIR（DIR 已在硬件接地）。

### 7.7 单通道轮询采样 — 标准七步

与 GPIO「开时钟 → 设模式」对应，最小可运行流程：

```text
① 开时钟（GPIOA + ADC），配置 ADC 时钟分频
② PA 口设为 GPIO_MODE_ANALOG
③ adc_deinit → 右对齐 → 规则组长度 1 → 软件触发
④ adc_enable → 短延时 → adc_calibration_enable
⑤ 每次采样：adc_regular_channel_config(rank, channel, sample_time)
⑥ adc_software_trigger_enable → 等待 ADC_FLAG_EOC → 清标志
⑦ adc_regular_data_read() 得 raw
```

```c
rcu_periph_clock_enable(RCU_GPIOA);
rcu_periph_clock_enable(RCU_ADC);
rcu_adc_clock_config(RCU_ADCCK_APB2_DIV6);                        /* ① */

gpio_mode_set(GPIOA, GPIO_MODE_ANALOG, GPIO_PUPD_NONE,
              GPIO_PIN_3 | GPIO_PIN_4);                          /* ② */

adc_deinit();
adc_data_alignment_config(ADC_DATAALIGN_RIGHT);
adc_channel_length_config(ADC_REGULAR_CHANNEL, 1U);
adc_external_trigger_source_config(ADC_REGULAR_CHANNEL, ADC_EXTTRIG_REGULAR_NONE);
adc_external_trigger_config(ADC_REGULAR_CHANNEL, DISABLE);
adc_enable();                                                     /* ④ */

for (volatile uint32_t i = 0U; i < 10000U; i++) { }             /* enable 后需稳定一段时间 */
adc_calibration_enable();

/* ⑥⑦ 读 CH3 示例 */
adc_regular_channel_config(0U, ADC_CHANNEL_3, ADC_SAMPLETIME_55POINT5);
adc_software_trigger_enable(ADC_REGULAR_CHANNEL);
while (RESET == adc_flag_get(ADC_FLAG_EOC)) { }
adc_flag_clear(ADC_FLAG_EOC);
uint16_t raw = adc_regular_data_read();
```

多通道时 **每次改通道** 都要重新 `adc_regular_channel_config` 再触发；或改用 §7.12 TIMER 触发 + DMA。

### 7.12 TIMER 触发 + DMA + 中断（完整链路）

与 [§4.10](#410-中心对齐-pwm--trgo-触发-adc电流环同步) 配合使用：**TIMER2 持续输出 PWM 并发出 TRGO → ADC 扫描采样 → DMA 写入内存 → 中断里处理**。

#### 7.12.1 整体时序

```text
  TIMER2 PWM 周期
  ────┐     ┌────┐     ┌────
      └─────┘    └─────┘
            ↑ TRGO（UPDATE）
            │
      ADC 扫描 CH3 → CH4
            │
      DMA 写入 buf[0], buf[1]
            │
      ADC EOC 中断（或 DMA FTF）
            │
      电流环 PI → 更新 CCR（影子，下周期生效）
```

**本板一次触发得到：** `buf[0]` = PA3 电流 raw，`buf[1]` = PA4 电压 raw。

#### 7.12.2 涉及外设与文件

| 外设 | 库文件 | 作用 |
|------|--------|------|
| TIMER2 | `gd32f1x0_timer.c` | 中心对齐 PWM + TRGO |
| ADC | `gd32f1x0_adc.c` | 扫描、外部触发、DMA 请求 |
| DMA | `gd32f1x0_dma.c` | 搬运 `ADC_RDATA` → 内存 |
| NVIC | `gd32f1x0_misc.c` | 开 `ADC_CMP_IRQn` 或 `DMA_Channel0_IRQn` |
| SYSCFG | `gd32f1x0_syscfg.c` | 可选：ADC DMA 映射到 CH1 |

#### 7.12.3 初始化顺序

```text
① bsp_adc GPIO + ADC 时钟/校准（§7.7 前半）
② ADC：扫描 2 通道、外部触发 T2_TRGO、使能 DMA
③ DMA：CH0 循环模式，外设 = ADC_RDATA，内存 = adc_buf[]
④ NVIC：使能 ADC EOC 中断（或 DMA FTF）
⑤ TIMER2：中心对齐 PWM + TRGO + CCR 影子（§4.10.5）
⑥ timer_enable → 自动开始周期性采样
```

**顺序要点：** ADC/DMA/NVIC 就绪后再 `timer_enable()`，避免首次 TRGO 丢失。

#### 7.12.4 ADC 侧配置

```c
#define ADC_DMA_BUF_LEN    2U    /* 扫描 2 通道：CH3 + CH4 */
static uint16_t adc_dma_buf[ADC_DMA_BUF_LEN];

/* §7.7 已完成：GPIO 模拟、时钟、校准、adc_enable */

adc_special_function_config(ADC_SCAN_MODE, ENABLE);
adc_channel_length_config(ADC_REGULAR_CHANNEL, 2U);
adc_regular_channel_config(0U, ADC_CHANNEL_3, ADC_SAMPLETIME_55POINT5);
adc_regular_channel_config(1U, ADC_CHANNEL_4, ADC_SAMPLETIME_55POINT5);

adc_external_trigger_source_config(ADC_REGULAR_CHANNEL, ADC_EXTTRIG_REGULAR_T2_TRGO);
adc_external_trigger_config(ADC_REGULAR_CHANNEL, ENABLE);

adc_dma_mode_enable();
adc_interrupt_enable(ADC_INT_EOC);
```

| 配置项 | 值 | 说明 |
|--------|-----|------|
| 扫描模式 | ENABLE | 一次触发依次采 CH3、CH4 |
| 规则组长度 | 2 | rank0=CH3，rank1=CH4 |
| 触发源 | `ADC_EXTTRIG_REGULAR_T2_TRGO` | 接 TIMER2 TRGO |
| DMA | ENABLE | 每完成一次转换自动搬运 |

#### 7.12.5 DMA 侧配置

GD32F130 默认 **ADC DMA 请求 → DMA CH0**。若 CH0 已被占用，可 `syscfg_dma_remap_enable(SYSCFG_DMA_REMAP_ADC)` 改映射到 **CH1**（见 `gd32f1x0_syscfg.h`）。

```c
dma_parameter_struct dma_init_struct;

rcu_periph_clock_enable(RCU_DMA);
dma_deinit(DMA_CH0);
dma_struct_para_init(&dma_init_struct);

dma_init_struct.periph_addr  = (uint32_t)&ADC_RDATA;
dma_init_struct.periph_width = DMA_PERIPHERAL_WIDTH_16BIT;
dma_init_struct.periph_inc   = DMA_PERIPH_INCREASE_DISABLE;
dma_init_struct.memory_addr  = (uint32_t)adc_dma_buf;
dma_init_struct.memory_width = DMA_MEMORY_WIDTH_16BIT;
dma_init_struct.memory_inc   = DMA_MEMORY_INCREASE_ENABLE;
dma_init_struct.number       = ADC_DMA_BUF_LEN;
dma_init_struct.direction    = DMA_PERIPHERAL_TO_MEMORY;
dma_init_struct.priority     = DMA_PRIORITY_HIGH;

dma_init(DMA_CH0, &dma_init_struct);
dma_circulation_enable(DMA_CH0);
dma_channel_enable(DMA_CH0);
```

| 参数 | 说明 |
|------|------|
| 外设地址 | `&ADC_RDATA`，固定不变 |
| 宽度 | 16-bit（与 ADC 右对齐 12-bit 一致） |
| 循环模式 | ENABLE，持续覆盖 `adc_dma_buf` |
| 传输数量 | 2（与扫描通道数相同） |

> **双缓冲进阶：** 缓冲长度设为 4（两组 I/V），开 DMA **HTF + FTF** 中断，一组处理时另一组采样。

#### 7.12.6 中断配置

**方案 A — ADC EOC 中断（推荐起步）**

扫描 2 通道完成后产生一次 EOC，与 PWM 周期一一对应，适合电流环。

```c
nvic_irq_enable(ADC_CMP_IRQn, 0U, 0U);   /* GD32F130：ADC 与 CMP 共用中断 */
```

**方案 B — DMA FTF 中断**

适合更长缓冲、双缓冲；需在 `gd32f1x0_it.c` 写 `DMA_Channel0_IRQHandler`。

```c
dma_interrupt_enable(DMA_CH0, DMA_INT_FTF);
nvic_irq_enable(DMA_Channel0_IRQn, 0U, 0U);
```

#### 7.12.7 中断服务函数（模板）

`gd32f1x0_it.c` 中实现。**只做电流环 + 写 CCR**；速度环降频；位置环与 I2C 在慢任务。

```c
void ADC_CMP_IRQHandler(void)
{
    if (RESET != adc_interrupt_flag_get(ADC_INT_FLAG_EOC)) {
        adc_interrupt_flag_clear(ADC_INT_FLAG_EOC);

        uint16_t i_raw = adc_dma_buf[0];   /* CH3 电流 */
        uint16_t v_raw = adc_dma_buf[1];   /* CH4 电压 */

        /* 1. raw → i_bus / v_bus（标定公式见 M-02 笔记） */
        /* 2. 电流环 PI：i_target → duty */
        /* 3. motor_set_duty(duty) 或写 TIMER2 CCR（§4.10 影子） */
        /* 4. 可选：过流/欠压判断 → motor_disable() */
    }
}
```

**ISR 约束：**

- 保持 **短**（目标 < 50 µs @ 10 kHz 控制周期）
- **禁止** I2C 读 AS5600
- 速度/位置环每 N 次中断执行一次，或放 SysTick

#### 7.12.8 `board_config.h` 建议宏

```c
#define ADC_PWM_TIMER             TIMER2
#define ADC_SAMPLE_RATE_HZ        20000U
#define ADC_PWM_PERIOD_CA         1799U   /* 中心对齐 20 kHz @ 72 MHz */
#define ADC_DMA_CHANNEL           DMA_CH0
#define ADC_DMA_BUF_LEN           2U
```

#### 7.12.9 调试速查

| 现象 | 排查 |
|------|------|
| `adc_dma_buf` 不更新 | TIMER2 是否 enable；TRGO 是否 UPDATE；ADC 外部触发是否 ENABLE |
| 只有 buf[0] 变 | 扫描长度是否为 2；rank0/rank1 通道是否正确 |
| 中断不进 | `ADC_CMP_IRQn` 是否使能；是否 `adc_interrupt_enable(ADC_INT_EOC)` |
| 采样率 2 倍于预期 | 中心对齐 UPDATE 频率；示波器测 TRGO |
| DMA 无请求 | `adc_dma_mode_enable()`；CH0 是否被占用；查 SYSCFG remap |
| 控制振荡 | 电流环 Kp/Ki 过大；CCR 未开影子；采样不在谷底 |

应用层换算、滤波、三环分频见 [M-02-M03 感知采集实现笔记 §4](./M-02-M03-感知采集实现笔记.md) 及后续 M-04/M-05 任务。

### 7.9 BSP 示例：`bsp_adc.c/h`

**`board_config.h`（引脚宏）：**

```c
#define ADC_CURRENT_PORT          GPIOA
#define ADC_CURRENT_PIN           GPIO_PIN_3
#define ADC_CURRENT_CHANNEL       ADC_CHANNEL_3

#define ADC_VBUS_PORT             GPIOA
#define ADC_VBUS_PIN              GPIO_PIN_4
#define ADC_VBUS_CHANNEL          ADC_CHANNEL_4
```

**`bsp_adc.h`：**

```c
#ifndef BSP_ADC_H
#define BSP_ADC_H
#include "gd32f1x0.h"

void bsp_adc_init(void);
uint16_t bsp_adc_read_channel(uint8_t channel);

#endif
```

**`bsp_adc.c`（核心逻辑）：**

```c
#include "bsp_adc.h"
#include "board_config.h"

void bsp_adc_init(void)
{
    rcu_periph_clock_enable(RCU_GPIOA);
    rcu_periph_clock_enable(RCU_ADC);
    rcu_adc_clock_config(RCU_ADCCK_APB2_DIV6);

    gpio_mode_set(ADC_CURRENT_PORT, GPIO_MODE_ANALOG, GPIO_PUPD_NONE, ADC_CURRENT_PIN);
    gpio_mode_set(ADC_VBUS_PORT, GPIO_MODE_ANALOG, GPIO_PUPD_NONE, ADC_VBUS_PIN);

    adc_deinit();
    adc_data_alignment_config(ADC_DATAALIGN_RIGHT);
    adc_channel_length_config(ADC_REGULAR_CHANNEL, 1U);
    adc_external_trigger_source_config(ADC_REGULAR_CHANNEL, ADC_EXTTRIG_REGULAR_NONE);
    adc_external_trigger_config(ADC_REGULAR_CHANNEL, DISABLE);
    adc_enable();

    for (volatile uint32_t i = 0U; i < 10000U; i++) { }
    adc_calibration_enable();
}

uint16_t bsp_adc_read_channel(uint8_t channel)
{
    adc_regular_channel_config(0U, channel, ADC_SAMPLETIME_55POINT5);
    adc_software_trigger_enable(ADC_REGULAR_CHANNEL);

    while (RESET == adc_flag_get(ADC_FLAG_EOC)) { }
    adc_flag_clear(ADC_FLAG_EOC);

    return adc_regular_data_read();
}
```

**`main.c` 最小调用（读 raw 验证）：**

```c
#include "bsp_adc.h"
#include "board_config.h"
#include "systick.h"

int main(void)
{
    uint16_t i_raw, v_raw;

    systick_config();
    bsp_adc_init();

    while (1) {
        i_raw = bsp_adc_read_channel(ADC_CURRENT_CHANNEL);
        v_raw = bsp_adc_read_channel(ADC_VBUS_CHANNEL);
        delay_1ms(10);
    }
}
```

`raw → i_bus / v_bus` 的标定系数、滑动平均滤波、过流保护属于应用层 `adc_monitor`，不在 BSP 内实现。

### 7.11 调试速查

| 现象 | 排查 |
|------|------|
| raw 全 0 或全 4095 | 引脚是否 `GPIO_MODE_ANALOG`；VDDA 是否 3.3 V；是否执行校准 |
| raw 不随输入变化 | 通道号是否与引脚一致（PA3=CH3）；是否等待 EOC |
| 读数跳动大 | 加长采样时间；应用层加滑动平均；检查前端 RC 滤波 |
| 校准后仍偏差大 | 正常，需应用层按硬件分压/运放标定 |
| 使能后立即校准失败 | `adc_enable()` 与 `adc_calibration_enable()` 之间加短延时 |
| PA4 配成 GPIO 输出 | 本板 PA4 必须保持模拟输入，不能当数字 IO |
