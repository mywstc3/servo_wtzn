## 使用说明

### 如何更新任务状态

在 **§2 任务总表** 中，直接修改对应行的「状态」列。可选值见下表，**仅使用这五种**，便于筛选与统计。

| 状态 | 含义 |
|------|------|
| 未开始 | 尚未着手 |
| 准备中 | 方案/环境/依赖已就绪，即将开发 |
| 推进中 | 正在实现或联调 |
| 已完成 | 验收通过 |
| 优化中 | 功能可用，持续改进性能或鲁棒性 |

### 如何添加新任务

1. 在 **§2 任务总表** 新增一行（大类 / 任务 / 子任务 三选一填「层级」列）。
2. 编号规则：电机类 `M-xx` / `M-xx-y`；通信类 `C-xx` / `C-xx-y`。
3. 在 **§3 任务详情** 对应分类下，复制文末 **附录 A · 任务详情模板** 粘贴并填写。
4. 子任务在父任务章节内追加 **子任务块**（见附录 B），并在总表中增加一行且「详情」列链接到子任务锚点。
5. 更新 **§4 依赖关系** 与 **§5 推荐实施计划**（如有影响）。

### 文档结构

```text
§1 使用说明          ← 你在这里
§2 任务总表          ← 一览所有大/小/子任务 + 状态 + 跳转
§3 任务详情          ← 每项任务的目标、思路、代码、问题、改进、验收、完成情况
§4~§6 架构与计划     ← 参考信息
附录 A/B             ← 复制粘贴模板
```

#### M-01 电机直接驱动

| 字段 | 内容 |

| **任务目标** | 不依赖闭环，按占空比/方向直接驱动 H 桥，实现开环正反转与关断 |
| **实现思路** | 按硬件说明 §8.3 配置 PA7/PB1 为 TIMER PWM（约 20 kHz），PA6 为 #SD 使能；正转时 PA7 输出 PWM、PB1 低；反转时 PB1 输出 PWM、PA7 低；上电默认 #SD 拉低关断 |
| **代码实现跳转** | 目标：[`servo_project/User/board_config.h`](../servo_project/User/board_config.h)（待建）、[`motor.c`](../servo_project/User/motor.c)（待建）、[`motor.h`](../servo_project/User/motor.h)（待建） · 参考例程：[`gpio_output/User/motor.c`](../GD32F130例程共32个/gpio_output/gpio_output/User/motor.c)（引脚需修正） |

| **方案改进** | 以硬件说明为准重写 `board_config.h`；查 GD32F130 手册确认 PB1 的 TIMER 通道与 AF 编号后再初始化 PWM |

## 4. 总体架构

```text
┌─────────────────────────────────────────────────────────┐
│  通信层（PA2 单线半双工 USART）                            │
│  命令下发 │ 状态上报 │ 参数读写                            │
└──────────────────────────┬──────────────────────────────┘
                           │
┌──────────────────────────▼──────────────────────────────┐
│  应用层：模式管理（开环 / 位置 / 速度 / 力矩）              │
└──────────────────────────┬──────────────────────────────┘
                           │
        ┌──────────────────┼──────────────────┐
        ▼                  ▼                  ▼
   位置环 PID          速度环 PID          电流环 PI
        │                  │                  │
        └──────────────────┴──────────────────┘
                           ▼
                    PWM 输出 + #SD 使能
                           │
        ┌──────────────────┼──────────────────┐
        ▼                  ▼                  ▼
     AS5600              ADC PA3            ADC PA4
     (I2C)               电流               母线电压
```

**目标控制环：** 位置环 → 速度环 → 电流(力矩)环 → H 桥 PWM
